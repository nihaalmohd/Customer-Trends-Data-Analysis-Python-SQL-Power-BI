# Customer-Trends-Data-Analysis-Python-SQL-Power-BI
📌 Project Overview

This project focuses on analyzing customer shopping patterns to uncover key business insights and enhance decision-making using data analytics. It combines SQL, Python, and Power BI with data-driven insights to simulate a real-world retail analytics solution.  The goal is to understand customer behavior, purchasing trends, and revenue drivers.

🎯 Objectives
-Analyze customer purchase behavior and trends
-Identify high-value customers and key revenue segments
-Perform data cleaning and transformation using SQL & Python
-Build interactive dashboards for business stakeholders
-Integrate Generative AI to generate automated insights and summaries
